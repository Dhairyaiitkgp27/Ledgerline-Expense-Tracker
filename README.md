## Overview

**Ledgerline** is a personal-finance dashboard built with React. Log your income and expenses, watch your spending break down by category, set monthly budgets, and get plain-language insights about your money — all in a clean, responsive interface that comes pre-loaded with sample data.

## ✨ Features

**Tracking**
- ➕ Add, edit and delete income & expense transactions
- 🗂️ Organised by category, date and note
- 📅 Month-by-month navigation

**Visualisation & Insights**
- 📊 6-month income-vs-spending bar chart
- 🍩 Category-breakdown donut chart
- 🎯 Per-category budgets with progress bars that flag overspending
- 💡 Auto-generated insights (top category, savings rate, month-end projection)

**Export**
- 📤 Download your data as a **CSV** spreadsheet
- 🖨️ Print a clean one-page **PDF** report

**Design**
- 📱 Fully responsive dashboard layout
- 🔢 Finance-grade, tabular number formatting
